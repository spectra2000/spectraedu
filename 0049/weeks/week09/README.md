<h2>09. Harici Kütüphaneler ve API Tabanlı Uygulamalar</h2>

<h3>Dersin Amacı</h3>
<p>
Bu haftada öğrenciler, Python ve Streamlit uygulamalarında harici kütüphaneleri
kullanmayı ve dış veri kaynaklarıyla (API’ler) etkileşim kurmayı öğreneceklerdir.
Amaç, paket programların gerçek dünya verileriyle nasıl beslendiğini kavramaktır.
</p>

<h3>Ders Kaynakları</h3>
<ul>
    <li>Python paket yönetimi (pip) ders notları</li>
    <li>Harici kütüphane ve API kullanımına giriş dokümanı</li>
    <li>Ders Slaytları</li>
</ul>

<h3>Görsel Kaynaklar</h3>
<ul>
    <li>API mantığı ve REST yapısı anlatım videosu</li>
    <li>Streamlit ile API kullanımı demo videosu</li>
</ul>

<h3>Dış Kaynaklar</h3>
<ul>
    <li>REST API kavramı</li>
    <li>JSON veri yapısı</li>
    <li>Açık veri servisleri (hava durumu, döviz, kripto vb.)</li>
</ul>

<h3>Uygulamalar</h3>
<ul>
    <li><code>requests</code> kütüphanesi ile API çağrısı yapma</li>
    <li>API’den gelen JSON verisini işleme</li>
    <li>Verinin Streamlit arayüzünde gösterilmesi</li>
    <li>Kullanıcı girdisine göre API isteği oluşturma</li>
    <li>
        Kullanılan araçlar ve kütüphaneler:
        Python, Streamlit, <code>requests</code>, Pandas
    </li>
</ul>

<h3>Ödev</h3>
<ul>
    <li>
        <strong>Ödev 9:</strong>
        Streamlit kullanarak bir açık API’den veri alan
        ve bu veriyi kullanıcı arayüzünde gösteren
        bir paket program geliştiriniz.
        Uygulama, en az bir kullanıcı girdisine göre
        API isteği oluşturmalıdır.
    </li>
</ul>
