## A. Dersin Amacı

Bu haftada öğrenciler, dönem boyunca
öğrendikleri R, Shiny ve paket program
kavramlarını kullanarak
kapsamlı bir **final projesi** geliştirecektir.

Amaç, öğrencinin
bir paket programı
baştan sona tasarlayıp
uygulayabildiğini göstermesidir.
