<h2>10. API Tabanlı Uygulama (Gelişmiş)</h2>

<h3>Dersin Amacı</h3>
<p>
Bu haftada öğrenciler, API tabanlı Streamlit uygulamalarını bir adım ileri taşıyarak
kullanıcı girdileriyle çalışan, hata yönetimi olan ve daha gerçekçi senaryolara
uygun paket programlar geliştirmeyi öğreneceklerdir.
</p>

<h3>Ders Kaynakları</h3>
<ul>
    <li>Gelişmiş API kullanımı ders notları</li>
    <li>Streamlit hata yönetimi ve durum göstergeleri dokümanı</li>
    <li>Ders Slaytları</li>
</ul>

<h3>Görsel Kaynaklar</h3>
<ul>
    <li>Gelişmiş API senaryoları anlatım videosu</li>
    <li>Streamlit spinner ve hata yönetimi demo videosu</li>
</ul>

<h3>Dış Kaynaklar</h3>
<ul>
    <li>REST API hata yönetimi yaklaşımları</li>
    <li>Python <code>try / except</code> kullanımı</li>
    <li>Açık API servisleri dokümantasyonları</li>
</ul>

<h3>Uygulamalar</h3>
<ul>
    <li><code>st.text_input()</code> ile kullanıcıdan veri alma</li>
    <li><code>requests.get()</code> ile dinamik API çağrısı yapma</li>
    <li><code>try / except</code> blokları ile hata yakalama</li>
    <li><code>st.spinner()</code> ile yükleniyor göstergesi kullanma</li>
    <li>
        Kullanıcı girdisine göre API isteği oluşturan,
        hata yönetimi içeren bir Streamlit uygulaması geliştirme
    </li>
    <li>
        Kullanılan araçlar ve kütüphaneler:
        Python, Streamlit, <code>requests</code>, Pandas
    </li>
</ul>

<h3>Ödev</h3>
<ul>
    <li>
        <strong>Ödev 10:</strong>

        mesajlarla bildirmelidir.
    </li>
</ul>
