# Hafta 10 – API Tabanlı Uygulama
**Ders:** Paket Programlar (Python / Streamlit Framework)  
**Hafta:** 10  
**Kapsam:** API kavramı ve dış veri kullanımı  

---

## 1. Dersin Amacı

Bu haftanın amacı, Streamlit uygulamalarında
dış servislerden veri almayı sağlayan
API kavramını tanıtmak ve
bu verilerin uygulamalara nasıl entegre edildiğini açıklamaktır.

Bu hafta sonunda öğrenciler:
- API kavramını açıklar
- API ile veri almanın mantığını kavrar
- Dış verilerle çalışan uygulamalar geliştirir

---

## 2. API Nedir?

**API (Application Programming Interface)**,
bir yazılımın başka bir yazılımla
iletişim kurmasını sağlayan arayüzdür.

API’ler sayesinde:
- Uygulamalar birbirleriyle veri paylaşır
- Dış servislerin sunduğu bilgiler kullanılabilir

---

## 3. API Kullanım Senaryoları

API’ler genellikle:
- Hava durumu verileri
- Döviz kurları
- Harita ve konum servisleri
- Sosyal medya verileri

gibi birçok alanda kullanılır.

Bu senaryolar, paket programları
daha işlevsel hâle getirir.

---

## 4. API ile Veri Almanın Genel Mantığı

Bir API ile çalışırken:
- Belirli bir istekte bulunulur
- Sunucu bu isteğe yanıt verir
- Gelen veri uygulama içinde işlenir

Bu süreç, istemci–sunucu mantığına dayanır.

---

## 5. API Yanıtları ve Veri Formatları

API’ler genellikle:
- Yapılandırılmış veri formatları
- Kolay işlenebilir çıktılar

sunar.

Bu yapı, verilerin
uygulamalar tarafından rahatça kullanılmasını sağlar.

---

## 6. Streamlit ve API Entegrasyonu

Streamlit:
- API’den alınan verileri
- Hızlıca arayüzde göstermeyi sağlar

Bu sayede:
- Canlı verilerle çalışan uygulamalar
- Gerçek zamanlıya yakın deneyimler

oluşturulabilir.

---

## 7. API Kullanımında Dikkat Edilmesi Gerekenler

API kullanırken:
- Hata durumları göz önünde bulundurulmalıdır
- Yanıt alınamazsa uygun mesajlar gösterilmelidir

Bu yaklaşım:
- Uygulamanın güvenilirliğini artırır

---

## 8. Paket Programlarda API’nin Önemi

Paket programlar:
- Statik verilerle sınırlı kalmaz
- Sürekli güncellenen bilgiler sunar

API’ler, bu dinamizmi sağlar.

---
