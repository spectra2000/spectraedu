## 1. Dersin Amacı

Bu haftanın amacı, web sayfalarının görsel tasarımını sağlayan CSS dilini
tanıtmak ve HTML içeriklerinin nasıl stillendirildiğini açıklamaktır.

Bu hafta sonunda öğrenciler:

- HTML ve CSS arasındaki farkı,
- Responsive tasarımın temel prensiplerini öğrenecektir.

---

## 2. CSS Nedir?

**CSS (Cascading Style Sheets)**, web sayfalarının görünümünü tanımlayan
bir stil dilidir.

CSS:
- Renkleri, yazı tiplerini ve yerleşimi kontrol eder
- HTML yapısından bağımsızdır
- Sayfa tasarımını merkezi olarak yönetmeyi sağlar

HTML yapıysa, CSS görünümüdür.
