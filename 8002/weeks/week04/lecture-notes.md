# Hafta 04 – HTML Temelleri
**Ders:** Internet Teknolojileri ve Web Programlama  
**Hafta:** 4

---

## 1. Dersin Amacı

Bu haftanın amacı, web sayfalarının temelini oluşturan HTML dilini tanıtmak
ve bir web sayfasının yapısal olarak nasıl oluşturulduğunu açıklamaktır.

Bu hafta sonunda öğrenciler:
- HTML’in ne işe yaradığını,
- Bir HTML dokümanının yapısını,
- Temel HTML etiketlerini kullanmayı öğrenecektir.

---

## 2. HTML Nedir?

**HTML (HyperText Markup Language)**, web sayfalarının yapısını tanımlayan
bir işaretleme dilidir.

HTML:
- Bir programlama dili değildir
- Sayfanın içeriğini ve yapısını tanımlar
- Tarayıcılar tarafından yorumlanır

HTML, web’in iskeletidir.

---

## 3. HTML Doküman Yapısı

Temel bir HTML dokümanı şu yapıya sahiptir:

```html
<!DOCTYPE html>
<html>
  <head>
    <title>Sayfa Başlığı</title>
  </head>
  <body>
    İçerik buraya yazılır
  </body>
</html>
