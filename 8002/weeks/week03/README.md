<h2>03. İnternetin Yapısı ve HTTP Temelleri</h2>

<h3>Dersin Amacı</h3>
<p>
Bu haftanın sonunda öğrenciler; internetin nasıl çalıştığını kavrayacak,
istemci–sunucu (client–server) mimarisini anlayacak,
HTTP protokolünün temel mantığını öğrenecek
ve tarayıcı–sunucu arasındaki iletişimi kavramsal düzeyde açıklayabilecektir.
</p>

<h3>Ders Kaynakları</h3>
<ul>
    <li>Hafta içeriği ders notları</li>
    <li>HTTP ve İnternet temel kavramları sunumu</li>
</ul>

<h3>Görsel Kaynaklar</h3>
<ul>
    <li>HTTP Request–Response akışını gösteren tanıtım videoları</li>
    <li>Client–Server mimarisini anlatan şematik görseller</li>
</ul>

<h3>Dış Kaynaklar</h3>
<ul>
    <li>Internet ve Web arasındaki farklar</li>
    <li>HTTP protokolü hakkında temel okuma materyalleri</li>
</ul>

<h3>Uygulamalar</h3>
<ul>
    <li>Bir web sitesinin tarayıcı üzerinden nasıl yüklendiğinin incelenmesi</li>
    <li>Tarayıcı geliştirici araçları (Developer Tools) ile Network sekmesinin kullanımı</li>
    <li>HTTP istek ve cevaplarının gözlemlenmesi</li>
</ul>

<h3>Ödev</h3>
<ul>
    <li>
        <strong>Ödev 3 (Opsiyonel):</strong>
        Bir web sitesinin Network trafiğini inceleyerek en az 3 farklı HTTP isteğini gözlemleyin
        ve bu isteklere ait durum (status) kodlarını not alın.
    </li>
</ul>
