# Hafta 06 – Modern Web Ekosistemi
**Ders:** Internet Teknolojileri ve Web Programlama  
**Hafta:** 6  
**Kapsam:** Frontend–backend ayrımı, framework kavramı ve modern web mimarileri  

---

## 1. Dersin Amacı

Bu haftanın amacı, modern web uygulamalarının nasıl geliştirildiğini
yüksek seviyeden ele almak ve web ekosistemindeki temel kavramları
tanıtmaktır.

Bu hafta sonunda öğrenciler:
- Frontend ve backend kavramlarını ayırt edebilir,
- Framework ve kütüphane farkını anlayabilir,
- Modern web geliştirme yaklaşımlarını tanıyacaktır.

---

## 2. Frontend ve Backend Kavramları

### 2.1 Frontend Nedir?

Frontend:
- Kullanıcının gördüğü arayüzdür
- Tarayıcı üzerinde çalışır
- HTML, CSS ve JavaScript kullanır

Örnek görevler:
- Sayfa tasarımı
- Kullanıcı etkileşimleri
- Formlar ve animasyonlar

---

### 2.2 Backend Nedir?

Backend:
- Sunucu tarafında çalışan sistemdir
- İş mantığını ve veri yönetimini içerir

Örnek görevler:
- Veritabanı işlemleri
- Kimlik doğrulama
- API servisleri

---

## 3. Full-Stack Geliştirme

**Full-stack geliştirici**, hem frontend hem de backend tarafında
çalışabilen geliştiricidir.

Bu yaklaşım:
- Küçük ekiplerde yaygındır
- Projenin tamamına hâkim olmayı sağlar

---

## 4. Framework ve Kütüphane Farkı

### Kütüphane (Library)
- Belirli bir problemi çözer
- Geliştirici ne zaman kullanacağını kontrol eder

Örnek: jQuery

---

### Framework
- Uygulamanın genel yapısını belirler
- Akışı framework kontrol eder

Örnek: React, Angular, Svelte

> Framework “nasıl yazacağını” da belirler.

---

## 5. Modern Frontend Framework’leri

Günümüzde yaygın kullanılan bazı framework’ler:
- React
- Vue
- Angular
- Svelte

Bu framework’ler:
- Bileşen tabanlı yapı sunar
- Kod tekrarını azaltır
- Büyük projeleri yönetilebilir kılar

---

## 6. Backend Teknolojilerine Genel Bakış

Backend tarafında sık kullanılan teknolojiler:
- Node.js
- Python (Flask, Django)
- Java (Spring)
- PHP (Laravel)

Bu teknolojiler genellikle API tabanlı çalışır.

---

## 7. API Kavramı

**API (Application Programming Interface)**, farklı yazılımların
birbiriyle iletişim kurmasını sağlar.

Web uygulamalarında:
- Frontend → API → Backend
şeklinde bir yapı kullanılır.

---

## 8. Modern Web Uygulama Mimarisi

Modern web uygulamaları genellikle:
- Bileşen tabanlı
- API odaklı
- Ayrık (decoupled) mimariye sahiptir

Bu yapı:
- Ölçeklenebilirliği artırır
- Bakımı kolaylaştırır
