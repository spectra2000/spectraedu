## 2. Internet Nedir?

**Internet**, dünya genelinde birbirine bağlı bilgisayar ağlarının oluşturduğu
küresel bir iletişim altyapısıdır.

Internet’in temel özellikleri:
- Merkezi bir yapısı yoktur
- TCP/IP protokol ailesi üzerine kuruludur
- Veri iletimi paket anahtarlama mantığıyla gerçekleştirilir

Internet bir uygulama değil, **altyapıdır**.  
Web, e-posta, dosya aktarımı gibi servisler Internet üzerinde çalışır.

---

## 3. World Wide Web (WWW) Nedir?

**World Wide Web (WWW)**, Internet üzerinde çalışan ve bilgiye erişimi kolaylaştıran
bir bilgi paylaşım sistemidir.

- 1991 yılında **Tim Berners-Lee** tarafından geliştirilmiştir
- HTML, HTTP ve URL kavramlarına dayanır
- Web tarayıcıları aracılığıyla kullanılır

> Önemli: Internet ve Web aynı kavram değildir.

---

## 4. Internet ve WWW Arasındaki Farklar

| Internet | World Wide Web |
|--------|----------------|
| Küresel ağ altyapısı | Internet üzerinde çalışan servis |
| Donanım ve protokoller | İçerik ve uygulama katmanı |
| TCP/IP kullanır | HTTP, HTML, URL kullanır |

---

## 5. Web’in Tarihsel Gelişimi

### 5.1 Internet’in Temelleri (1960–1990)

- 1969: ARPANET projesi
- Paket anahtarlama fikrinin ortaya çıkışı
- Akademik ve askerî kullanım ağırlıklıdır

Bu dönemde Internet, genel kullanıma açık değildir.

---

### 5.2 World Wide Web’in Ortaya Çıkışı (1991)

- İlk web sunucusu ve tarayıcı geliştirildi
- Statik HTML sayfalar kullanıldı
- Bilgi küresel ölçekte erişilebilir hâle geldi

---

## 6. Web 1.0, Web 2.0 ve Web 3.0

### Web 1.0
- Statik içerik
- Kullanıcı yalnızca içerik tüketir
- Etkileşim yoktur

### Web 2.0
- Dinamik içerik
- Sosyal medya ve kullanıcı etkileşimi
- İçerik kullanıcılar tarafından üretilir

### Web 3.0 (Genel Bakış)
- Dağıtık mimari
- Blockchain ve akıllı sözleşmeler
- Veri sahipliği ön plandadır

---

## 7. Internet, Intranet ve Extranet

- **Internet:** Herkese açık küresel ağ
- **Intranet:** Kurum içi, kapalı ağ
- **Extranet:** Yetkilendirilmiş dış erişime açık ağ

Bu yapılar aynı teknolojileri kullanır; fark erişim ve güvenlik politikalarındadır.

---

## 8. İstemci – Sunucu (Client–Server) Mimarisi

Web tabanlı sistemler **istemci–sunucu mimarisi** ile çalışır.

- **İstemci (Client):** Web tarayıcısı
- **Sunucu (Server):** Web sunucusu

İstemci bir istek gönderir, sunucu bu isteğe yanıt verir.

---

## 9. IP Adresi Kavramı

IP adresi, Internet üzerindeki her cihazın sahip olduğu benzersiz sayısal kimliktir.

- **IPv4:** 32 bit adresleme
- **IPv6:** 128 bit adresleme

IP adresleri cihazların birbirini bulmasını sağlar.

---

## 10. DNS (Domain Name System)

**DNS**, alan adlarını IP adreslerine çeviren dağıtık bir isimlendirme sistemidir.

DNS sayesinde:
- İnsanlar kolay hatırlanabilir alan adları kullanır
- Bilgisayarlar sayısal IP adresleriyle iletişim kurar
