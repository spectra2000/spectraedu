## A. Dersin Amacı

bir arada tutmayı sağlayan diziler ve metin işlemlerinin
temelini oluşturan string kavramı ele alınacaktır.

Öğrenciler, çoklu veri yapılarıyla çalışmanın
mantığını kavrayacaktır.
