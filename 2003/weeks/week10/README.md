<h2>10. Diziler, Pointer İlişkisi ve String İşlemleri</h2>

<h3>Dersin Amacı</h3>
<p>
Bu haftada öğrenciler, C dilinde diziler ile pointer’lar
arasındaki güçlü ilişkiyi öğreneceklerdir. Özellikle
string (karakter dizileri) yapısının C dilinde nasıl
çalıştığı, bellek kullanımı ve sık yapılan hatalar
üzerinde durulacaktır.
</p>

<h3>Ders Kaynakları</h3>
<ul>
    <li>C diziler ve pointer ilişkisi ders notları</li>
    <li>String işlemleri ve karakter dizileri dokümanı</li>
    <li>Ders Slaytları</li>
</ul>

<h3>Görsel Kaynaklar</h3>
<ul>
    <li>Dizi–pointer bellek eşleşmesi anlatım videosu</li>
    <li>String yapısının bellekte gösterimi videosu</li>
</ul>

<h3>Dış Kaynaklar</h3>
<ul>
    <li>C dilinde string yapısı ve null karakter (<code>'\0'</code>)</li>
    <li><code>string.h</code> kütüphanesi referansı</li>
    <li>Buffer overflow ve string hataları makalesi</li>
</ul>

<h3>Uygulamalar</h3>
<ul>
    <li>Dizilerin fonksiyonlara pointer olarak gönderilmesi</li>
    <li>Karakter dizisi (string) tanımlama</li>
    <li><code>strlen</code>, <code>strcpy</code>, <code>strcmp</code> kullanımı</li>
    <li>String üzerinde döngü ile gezinme</li>
    <li>Hatalı string kullanım örnekleri</li>
</ul>

<h3>Bu Haftanın Odak Noktaları</h3>
<ul>
    <li>String’in aslında bir <strong>char dizisi</strong> olduğu gerçeği</li>
    <li>Null karakterin unutulması</li>
    <li>Yanlış boyutta string tanımlamaları</li>
    <li><code>scanf</code> ile string okurken yapılan hatalar</li>
    <li>Bellek taşması (buffer overflow) riskleri</li>
</ul>
