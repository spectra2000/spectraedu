<h2>05. Karar Yapıları (if, else, switch)</h2>

<h3>Dersin Amacı</h3>
<p>
Bu haftada öğrenciler, C dilinde program akışını kontrol eden
karar yapılarının nasıl çalıştığını öğreneceklerdir.
Koşullu ifadeler kullanarak farklı senaryolara göre
programın farklı yollar izlemesi sağlanacak, sık yapılan
mantık ve karşılaştırma hataları üzerinde durulacaktır.
</p>

<h3>Ders Kaynakları</h3>
<ul>
    <li>C dilinde karar yapıları ders notları</li>
    <li>if-else ve switch yapıları referans dokümanı</li>
    <li>Ders Slaytları</li>
</ul>

<h3>Görsel Kaynaklar</h3>
<ul>
    <li>Program akış diyagramları anlatım videosu</li>
    <li>if / else karar mekanizması örnek videosu</li>
</ul>

<h3>Dış Kaynaklar</h3>
<ul>
    <li>C karşılaştırma operatörleri tablosu</li>
    <li>Sık yapılan koşul yazım hataları makalesi</li>
    <li>switch-case kullanım rehberi</li>
</ul>

<h3>Uygulamalar</h3>
<ul>
    <li><code>if</code> ve <code>else</code> kullanımı</li>
    <li>İç içe (nested) karar yapıları</li>
    <li>Karşılaştırma operatörleri ile koşul oluşturma</li>
    <li><code>switch-case</code> yapısı ile çoklu durum kontrolü</li>
    <li><code>break</code> kullanımının önemi</li>
</ul>

<h3>Bu Haftanın Odak Noktaları</h3>
<ul>
    <li>= ve == operatörlerinin karıştırılması</li>
    <li>Yanlış koşul yazımından kaynaklanan mantık hataları</li>
    <li>Gereksiz karmaşık if bloklarından kaçınma</li>
    <li>switch yapısının doğru senaryolarda kullanımı</li>
    <li>Okunabilir ve sade karar yapıları yazımı</li>
</ul>

</ul>
